# ⚽ Club App — منصة النادي (متجر + أكاديمية + دفع إلكتروني)

تطبيق ويب متكامل بـ Flask يغطي: صفحة رئيسية، متجر إلكتروني للنادي، أكاديمية تدريب الناشئين،
ونظام دفع إلكتروني تجريبي (Mock) — مبني بنفس الهوية البصرية (الألوان/الخطوط) المتفق عليها.

## الهيكل

```
club_app/
├── app.py                # نقطة التشغيل + بيانات تجريبية أولية (Seed)
├── config.py
├── extensions.py
├── models.py               # User, Product, AcademyProgram, AcademyRegistration, SavedCard, Order
├── requirements.txt
├── Procfile / render.yaml   # جاهز للنشر على Render
├── auth/                     # تسجيل الدخول / حساب جديد
├── main/                      # الصفحة الرئيسية
├── store/                      # متجر النادي + السلة
├── academy/                     # برامج الأكاديمية والتسجيل
├── payment/                      # اختيار البطاقة وتأكيد الدفع (Mock)
├── templates/
└── static/css/style.css           # نظام التصميم الموحّد (ألوان Hex + خطوط Cairo/Tajawal)
```

## التشغيل محليًا

```bash
python -m venv venv
venv\Scripts\activate      # ويندوز
source venv/bin/activate    # ماك/لينكس

pip install -r requirements.txt
python app.py
```

يفتح على `http://127.0.0.1:5000`. البيانات التجريبية (منتجات + برامج أكاديمية) تُزرع تلقائيًا أول مرة.

## سير العمل

1. أنشئ حساب من "إنشاء حساب".
2. من "متجر النادي": أضف منتجات للسلة → "المتابعة للدفع" → اختر/أضف بطاقة → "تأكيد الدفع".
3. من "الأكاديمية": سجّل فبرنامج — إذا مجاني يتأكد فورًا، وإذا مدفوع يوديك لنفس صفحة الدفع.

## ⚠️ ملاحظة أمان مهمة بخصوص الدفع

نظام الدفع هنا **Mock تجريبي بالكامل** لأغراض العرض فقط:
- لا يتصل بأي بوابة دفع حقيقية.
- لا يُخزَّن رقم البطاقة الكامل، فقط آخر 4 أرقام (لمحاكاة العرض البصري).
- **لا يصلح إطلاقًا للاستخدام الفعلي بأموال حقيقية.**

للنشر الفعلي، يجب ربط بوابة دفع معتمدة جزائريًا (مثل **CIB / Edahabia** عبر SATIM)
أو عالميًا (Stripe)، والالتزام بمعايير PCI-DSS، وعدم تخزين أي بيانات بطاقة خام فقاعدة بياناتك مطلقًا.

## النشر على Render

نفس الطريقة المستخدمة فمشروعك السابق:
- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `gunicorn app:app --bind 0.0.0.0:$PORT`
- زد متغير بيئة `SECRET_KEY` بقيمة عشوائية.
